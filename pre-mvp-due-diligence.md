# Stock Dashboard Pre-MVP Due Diligence

작성일: 2026-04-29
목적: MVP 구현에 들어가기 전, 무료 1분 갱신 대시보드의 데이터 소스·안정성·약관/차단 리스크·운영 구조를 검증하고, 사용자에게 한 번에 물어볼 의사결정 질문을 정리한다.

## 검토 축

1. 네이버증권 모바일 데이터 수집 가능성
2. 한국투자증권 KIS / KRX / 국내 소스에서 한국 선물·야간선물·채권 가능성
3. yfinance/Binance 기반 1분 갱신 안정성
4. 장시간 기반 업데이트/상태 표시 설계
5. 무료 운영 한계와 fallback 전략

## 원칙

- MVP 직전까지는 성급히 구현하지 않는다.
- 깨질 가능성이 높은 지점을 먼저 찾는다.
- 질문은 중간중간 흩뿌리지 않고, 필요한 의사결정만 모아서 다니에게 한 번에 묻는다.
